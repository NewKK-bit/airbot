⚙️ 실시간 검색 설정

SerpApi는 브라우저에서 직접 호출할 수 없어(보안·CORS), 무료 개인 프록시를 한 번만 배포하면 모든 노선을 실시간으로 검색할 수 있어요. 모니터링 중인 노선은 프록시 없이도 매일 수집된 데이터로 검색됩니다.

  1. dash.cloudflare.com 가입/로그인 → Workers & Pages → Create Worker
  2. 저장소의 proxy/worker.js 내용을 통째로 붙여넣고 Deploy
  3. Worker → Settings → Variables and Secrets → Secret SERPAPI_KEY 추가
  4. 배포된 https://….workers.dev 주소를 아래에 붙여넣기
